FSD(FULL STACK DEVELOPMENT)

A full-stack developer is a developer or engineer who can build both a website's front and back end. The front end (the parts of a website a user sees and interacts with) and the back end (the behind-the-scenes data storage and processing) require different skill sets.

(WDD) WEB DESIGN AND DEVELOPMENT

Web designers prioritize the site's visual appeal and usability, focusing on layout, color schemes, and navigation that improve user experience.

DD(DISTRIBUTED DATABASE)

A distributed database is a database that runs and stores data across multiple computers, as opposed to doing everything on a single machine.
